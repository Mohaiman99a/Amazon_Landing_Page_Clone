# Amazon Clone

A front-end Amazon homepage clone built using HTML and CSS for learning and practice purposes.

## Features

* Responsive navigation bar
* Search bar UI
* Hero section
* Product category cards
* Footer section
* Built with pure HTML and CSS

## Used 

* HTML5
* CSS3
* Font Awesome
